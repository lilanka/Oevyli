# Oevyli

This is the repository of Oevyli language and compiler. The aim of
Oevyli is to create Python like language that generates fast
execution codes. Oevyli has it's own Intermediate language called
`OIL - Oevyli Intermediate Language`. Note that the project is
under development.

[The Adventure to Oevyli](https://lilanka.github.io/oevyli.html)
page contains the implementation details of the compiler.
